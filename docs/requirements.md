# Requirements

1. Allow user to create, update, delete tasks.
2. Each task should have a title, description, due date, priority and status (e.g pending, in progress, completed)
3. Users should be able to assign tasks to other users and set remainders for tasks.
4. System should support searching, filtering tasks based on various criteria (e.g priority, due date, assigned user).
5. Users should be able to mark tasks as completed and view their task history.
